---
title: 'Checkbox'
---

Extends `<input type="checkbox" />`, allows input styling.

```js
import { Checkbox } from 'minerva-ui';
```

```jsx live
() => {
  const [checked, setChecked] = useState(false);
  return (
    <Checkbox checked={checked} onChange={() => setChecked(!checked)}>
      Stay Logged In
    </Checkbox>
  );
};
```

## Disabled Checkbox

```jsx live
<Checkbox disabled>Stay Logged In</Checkbox>
```

## Customize Checkbox

```jsx live
<Stack horizontal>
  <Checkbox _checked={{ bg: 'red.700' }}>Checked Background Color</Checkbox>
  <Checkbox bg="#0076ff" checkColor="#0076ff" _checked={{ bg: '#fff' }}>
    Change Check Color
  </Checkbox>
</Stack>
```

## Props

Props related to the Checkbox component.

| Name       | Type    | Is Required | Default | Description                                       |
| ---------- | ------- | ----------- | ------- | ------------------------------------------------- |
| `checked`  | boolean | optional    | false   | Sets if box is checked                            |
| `onChange` | boolean | optional    | none    | Action to take when value of `checked` is changed |
